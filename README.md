# Agent Collaboration Guide

这是给 AI Agent 的协作手册。每次新对话开始时，请先阅读本仓库相关文件，了解我的工作偏好和历史踩坑经验。

## 使用方式

对话开头告诉 agent：
> 先读一下 https://github.com/Ericcckkk/agent-guide 里的相关文件，了解我的偏好和规则。

## 文件结构

| 文件 | 内容 |
|------|------|
| `general.md` | 通用协作原则（适用所有项目） |
| `frontend.md` | 前端开发相关规则 |
| `deployment.md` | 部署/CI-CD/运维相关 |
| `server-infra.md` | 我的服务器环境信息 |
| `coding-style.md` | 代码风格偏好 |

## 持续更新

任何对话中发现新的偏好或踩坑经验，都可以直接提 PR 或追加到对应文件。格式统一用 checklist 条目。
